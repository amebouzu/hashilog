              ABOUT

              走ログとは

              走ログは、サーキット走行を楽しむすべてのドライバーのためのタイム共有サービスです。
              愛車の改造内容・使用タイヤ・走行コンディションを一緒に記録することで、自分の上達を可視化し、同じ条件で走るライバルと比較できる環境を提供します。

              私たちの想い

              「あの人の○○秒はどんな車・どんなタイヤで出したのか?」
              サーキットを走るドライバーが必ず一度は気になる疑問。SNS や動画ではバラバラに散らばっているこの情報を、一箇所にまとめて検索可能にすることが走ログの目的です。

              タイムだけでなく、その背景にあるセットアップの歴史を共有することで、走り込み派のドライバーが「次の一手」を見つけるヒントになる。そんなコミュニティを目指しています。

              こんな方におすすめ

              • 月1回以上サーキットを走るアマチュアドライバー
              • 愛車のセットアップを記録・共有したい方
              • 走行会・タイムアタック競技に出場している方
              • 同じ車種・同じタイヤでベストタイムを比較したい方
              • これからサーキットデビューする初心者の方

              FEATURES

              主な機能

              🏎️

              愛車のスペックシート

              足回り・エンジン・駆動系・ブレーキ・外装・内装の6カテゴリで改造内容を詳細に記録。

              ⏱️

              タイム記録

              総合タイム + セクター + 最高速 + 天候 + 路面 + 気温/路温まで、走行条件を網羅的に保存。

              📷

              写真エビデンス

              タイミングモニター・車載動画のスクリーンショットを複数枚アップロード可能。信頼性のあるリザルトに。

              🏆

              フェアなランキング

              サーキット・タイヤ銘柄・車種・メーカー で絞り込み。同じ条件のライバルとフェアに比較。

              🏁

              国内サーキット網羅

              富士・鈴鹿・もてぎなどの国際サーキットからミニサーキットまで、日本の主要コースを網羅。

              📣

              SNSシェア

              X / Facebook / Instagram / Threads / LINE にワンクリックでベストタイムや愛車紹介をシェア。

              USE CASE

              利用シーン

              📅 走行会の前後

              走行会前に同じサーキット・同じ車種のベストタイムを参考に。走り終わったらすぐタイム投稿でベスト更新を共有。

              🛠 セットアップ変更時

              タイヤを履き替えた・足回りを調整した。改造内容と一緒にタイムを残せば、変更前後の効果が一目瞭然。

              🎯 タイムアタック競技

              筑波スーパーバトルなどの競技に向けて、シーズンを通じて自分の成長を記録。SNSで盛り上がりを発信。

              サーキット運営者の方へ

              走ログでは、各サーキットの公式運営者の方に 自施設ページの編集権限 を発行しています。
              走行会・スポーツ走行の告知、料金変更のお知らせ、レースイベントの周知などを、走ログユーザーに直接届けることができます。
